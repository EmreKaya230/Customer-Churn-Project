Project Overview

This project focuses on predicting customer attrition (churn) for a telecommunications provider to identify at-risk users and improve retention strategies.

Technical Implementation

Predictive Modeling: Utilizes Logistic Regression to perform binary classification on customer status.

Data Processing: Extensive use of Pandas and NumPy for data cleaning and feature engineering.

Evaluation: Focuses on identifying key drivers of churn, such as contract types and monthly charges.

Tech Stack

Language: Python.

Libraries: Scikit-learn, Pandas, NumPy, Matplotlib.
